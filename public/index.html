  • About Odhecaton

    Overview

    Odhecaton is a web application designed for community choral organizations.
    It is intended to make choral directors more effective by providing a platform to share practice aids, such as sheet music pdfs and video recordings, with choral members.

    Why?

    Many choral organizations lack a central online location for practice resources, or do not have a mobile-friendly platform. Odhecaton aims to solve this problem.

    Details & Documentation

    At the moment, Odhecaton uses the Google YouTube Data API for video support. The Petrucci Music Library API will be coming soon to extend PDF upload support.
    View documentation here.

    Trivia

    Odhecaton is named after the Odhecaton, the first book of polyphonic music ever to be printed using moveable type, in 1501 by Ottaviano Petrucci.
